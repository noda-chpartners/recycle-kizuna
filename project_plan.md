# 株式会社リサイクル絆 LP

## 1. Project Description
北海道札幌市の出張買取専門リサイクルショップ「株式会社リサイクル絆」のランディングページ。リサイクル買取（腕時計・ブランドバッグ・切手葉書・貴金属・遺品整理）と建築リフォーム（壁クロス張替え・床施工・屋根壁塗装・サイディング施工）の2事業を展開。1ページ完結のLPで、高額買取査定・生前整理・遺品整理・内装工事の見積もり依頼を促進する。
